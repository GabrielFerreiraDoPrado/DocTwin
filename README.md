# DocTwin

Esta aplicação tem o objetivo de agir como um comparador de documentos para análise de similaridade.

## Requirementos técnicos

### Grails

Este projeto utiliza a versão `5.3.5` do Grails.

## Como rodar o projeto

1. Com o Grails 5.3.5 instalado, acesse a pasta raiz do projeto
2. Execute o comando `gradlew bootRun`
